## POKEMON Dashboard [electron]

### Installation process :

- clone repo
- navigate to project folder
- install all dependencies with npm
- run electron with npm 


```bash
$ git clone https://github.com/RandolphG/Pokemon-Dashboard.git
$ cd project
$ npm i
$ npm run electron 
```
